## 🌟 **Project Overview**

**Aarkham** is a revolutionary **Zero Knowledge based Aadhar verifier flash loan arbitrage service** that transforms how blockchain applications handle KYC (Know Your Customer) processes. Our protocol solves the critical problem of decentralized identity verification while maintaining complete user privacy.

## 🚀 **Quick Start**

<div align="center">
<img src="https://readme-typing-svg.herokuapp.com?font=Fira+Code&size=20&duration=2000&pause=1000&color=FFD700&center=true&vCenter=true&width=400&lines=Get+Started+Now!;Easy+Setup;Fast+Deployment;Ready+in+Minutes" width="400">
</div>

### Prerequisites
```bash
# Install Node.js, Rust, and Foundry
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
curl -L https://foundry.paradigm.xyz | bash
```

### Installation
```bash
# Clone the repository
git clone https://github.com/your-username/aarkham.git
cd aarkham

# Install dependencies
npm install
cd flash-loan-agent && pip install -r requirements.txt
cd ../zkp && cargo build --release
```

### Environment Setup
```bash
# Copy environment template
cp .env.example .env.local
```

### Configure Environment Variables
Edit `.env.local` with your values:
```bash
# Blockchain RPC
RPC_URL=https://polygon-mainnet.infura.io/v3/YOUR_INFURA_PROJECT_ID

# Contract Addresses (replace with deployed addresses)
NEXT_PUBLIC_FLASH_LOAN_EXECUTOR_ADDRESS=0x...
NEXT_PUBLIC_AAVE_POOL_ADDRESS=0x794a61358D6845594F94dc1DB02A252b5b4814aD

# Wallet Connect
NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID=your_walletconnect_project_id
```

### Running the Application
```bash
# Start the development server
npm run dev

# In another terminal, start the flash loan agent
cd flash-loan-agent && python app.py

# Deploy contracts (optional)
cd contracts && forge script script/FlashLoan.s.sol --rpc-url $RPC_URL --broadcast
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
